## 慧享生活圈

**技术栈：**
`SpringBoot` `MySQL` `Redis` `Lua` `Kafka` `Caffeine` `LangChain4j` `MybatisPlus`

慧享生活圈是一个连接本地商家与消费者的服务平台，提供商家信息检索、限时抢购、智能客服等功能，助力商家精准触达用户，提升营销转化效率。平台支持高并发秒杀活动，并具备智能问答与推荐能力。

**核心技术实现：**

- **秒杀性能优化：**
  利用 **Redis + Lua** 封装库存预检与扣减逻辑，保证库存不超卖并实现 **一人一单**。
- **缓存优化与多级缓存策略：**
  使用 **Redis 逻辑过期** 解决热点 Key 问题，并结合 **Caffeine + Redis** 构建多级缓存架构，提高数据访问速度并降低 Redis 压力。
- **支付与订单并发及数据一致性保障：**
  通过 **乐观锁** 解决高并发支付与订单冲突问题；采用 **更新数据库 + 删除缓存** 策略，并结合 **消息队列重试机制** 与 **TTL** 保证数据最终一致性。
- **滑动窗口限流：**
  基于 **Redis + AOP** 实现全局、IP 与用户多维度限流，防止刷单、爬虫及系统过载。
- **智能客服系统：**
  接入 **阿里云百炼大模型**，Redis 支持会话记忆，通过 **Function Calling** 实现信息查询和预约到店功能。
- **秒杀流程优化：**
  将同步流程改造为 **消息队列异步处理**，订单生成后由 MQ 异步执行，提高高并发场景下系统吞吐量与响应速度。
